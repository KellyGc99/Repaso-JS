ARRAYS

También llamados arreglos, se identifican con llaves cuadradas [] , es recomendable utilizar CONST tanto en arrays como en objetos y demás datos , puesto que aunque su contenido se modifique, el dato seguirá siendo un arreglo. Un array es una colección de elementos de diferente tipo, dentro de un array puede haber un objeto, un boolena, otro array, etc dentro del array, los elemtnos tienen posiciones, empiezan desde el 0 pero este no es el mismo numero de datos que exitesn dentro de el es solo la posición en la que se encuentran los datos. La propiedad LENGHT nos devuelve la longitud del array. Podemos acceder a un dato del array por medio de la posición, esto solo sirve si la conocemos. Podemos crear un nuevo array a partir de la opción Array.of(esto hace parte de ECMAScript 6) , con Array.fill podemos asignar a x cantidad de datos un mismo valor. Para los array también existen métodos que nos permite interactuar con los datos del array como PUSH (agrega un elemnto) y POP (elimina el ultimo elemento del array) FOREACH (permite una función que se ejecuta por cada elemnto del array) existen más métodos que son para escribir programación funcional, más avanzados. Por ahora estos 3 métodos son básicos y permiten hacer modificaciones.
