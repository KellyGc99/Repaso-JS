VARIABLES:

Son lugares en la memoria de la computadora para almacenar datos, antes se
utilizaba la palabra VAR , sin embargo ahora utilizamos CONST o LET. VAR
no permitia la ejecució del código en bloque , le quitaba alcance pues
esto las definia como variables globales. VAR de ámbito global, ya no se
usa porque no es una buena práctica y además implica problemas en la
ejecución del código ya que aparecen en el objeto WINDOW , LET ámbito de
bloque, solp existe en el bloque donde ha sido definido. Un bloque en JS
es lo que está definido entre llaves {}


Var quedará en el objeto global window, mientras que let y const no Var
variables globales, Let variables de bloque.

No usar var, usar let o const dependiendo del caso, const es para declarar
variables que no cambiarán su valor, diferente de let que es para
variables modificables.

USO Y CARACTERISTICAS DE CONST
Cuando tengamos variables que no cambiaran su valor durante el uso de mi
aplicación. Además de eso si utilizamos CONST no podemos dejarla como una
variable vacia, cosa que si se puede con LET. Const no permite ser
modificada cuando se usan valores primitivos, es diferente al usar valores
compuestos como arrays, objetos , etc.
No puedes volver variable un valor constante.
